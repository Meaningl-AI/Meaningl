## Packages
framer-motion | Complex animations and page transitions
react-markdown | Rendering AI chat responses with markdown support
remark-gfm | GitHub Flavored Markdown plugin for react-markdown
react-syntax-highlighter | Syntax highlighting for code blocks in chat
eventsource-parser | robust parsing of SSE streams from the chat API

## Notes
Chat API uses Server-Sent Events (SSE) for streaming responses.
Voice input uses browser's native Web Speech API (webkitSpeechRecognition).
Theme defaults to "dark" with a futuristic neon aesthetic.
